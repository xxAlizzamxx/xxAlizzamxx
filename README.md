## 📱 Proyecto Destacado — Vidalink

**Vidalink** es una aplicación móvil de respuesta a emergencias diseñada para asistir a personas en situaciones críticas antes de la llegada de atención médica profesional.

### 🔹 Funcionalidades principales:
- 📍 Geolocalización en tiempo real mediante integración con mapas
- 🚨 Sistema de alertas que conecta con voluntarios cercanos con conocimientos en primeros auxilios
- 💬 Chat en tiempo real con el voluntario
- 🤖 Asistente con IA para orientación en primeros auxilios
- 🏥 Detección automática del hospital más cercano

### 🔹 Tecnologías utilizadas:
- Desarrollo en **Android Studio (Java)**
- Backend con **Firebase**
- Integración con **APIs de geolocalización y mapas**
- Base de datos en tiempo real

### 🔹 Estado del proyecto:
- Prototipo funcional en fase de pruebas
- Alto potencial de escalabilidad e impacto social
